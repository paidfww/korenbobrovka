GitHub Copilot

GitHub Copilot - код жазуды автоматтандыратын ЖИ-көмекші. VS Code, IntelliJ және басқа редакторлармен біріктіріледі.

Не істей алады: Функцияларды аяқтау, тесттер жазу, қателерді түзету.

📝 Менің тәжірибем

Мен ChatGPT және GitHub Copilot құралдарын қолданып көрдім. ChatGPT күрделі алгоритмдерді түсіндіруге көп көмектесті. GitHub Copilot кодымды жылдам жазуға мүмкіндік берді.

Қандай құрал көмектесті? ChatGPT ең көп көмектесті, әсіресе JavaScript функцияларын жазуда.

ЖИ нені дұрыс/ыңғайсыз жасады? Код құрылымын дұрыс ұсынды, бірақ кейде ескірген синтаксис берді. Мен оны қолмен түзеттім.

ЖИ 5 жылдан кейін? ЖИ әзірлеушілерді алмастырмайды, керісінше олардың жұмысын жеңілдетеді. Қарапайым тапсырмалар автоматтандырылады.
